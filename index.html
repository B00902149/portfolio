🏆 Final Year Project · COM668

Coaching Hub

A cross-platform mobile app that solves fitness "app fatigue" — consolidating workout tracking, nutrition logging, real-time health metrics, and a community newsfeed into a single, intuitive dashboard. Integrates with Apple Health, Google Fit, MyFitnessPal, ExerciseDB and more.

Developed using an adapted Agile/Scrum approach with 2-week sprints, GitHub Projects for tracking, and a full C4 architecture model. Targeted 90%+ task completion rate in usability testing.

React Native Expo Node.js MongoDB Atlas Redux Toolkit REST API JWT Auth Firebase NativeBase
View GitHub → ▶ View Website
Dashboard

Your command centre. Displays a personalised welcome, today's workout and nutrition summary, and quick-access tiles to every section of the app.

  • Daily quote card fetched from Bible API
  • Today's workout tile — pulls active workout plan from MongoDB
  • Calories logged summary from Nutrition module
  • Quick-nav tiles: My Health, Exercise, Nutrition, Community
  • Personalised greeting using authenticated user's name
My Health

A live metrics overview pulling real data from connected health APIs. Users can log, edit, and track key health indicators over time.

  • Weight, sleep, BPM, activity, and nutrition tiles
  • Data synced from Apple Health / Google Fit via Health Connect
  • Tap any metric to view historical entries and trends
  • Add new entries — stored in MongoDB with timestamp
  • Edit and delete individual records
Workouts

Structured workout tracking screen. Displays the active training plan with sets, reps, and weight logging for each exercise in the session.

  • Workout plan loaded dynamically from MongoDB
  • Section list: Warm Up and Main Exercises
  • Input fields for sets, reps, and weight per exercise
  • Play button navigates to Exercise Detail for video demo
  • Completion triggers rating + post to Community feed
Exercise Library

Searchable database of exercises. Filter by name, muscle group, or equipment type. Results show average user ratings pulled from MongoDB.

  • Real-time search bar with live filtering
  • Filter by muscle group: Push / Pull / Legs / Core / Cardio
  • Star ratings averaged across all user submissions
  • Each result links to full Exercise Detail screen
  • Add any exercise to a custom workout plan
Exercise Detail

In-depth view of an individual exercise. Includes a video demonstration, step-by-step instructions, and an option to add to your current workout.

  • Video demo via expo-av with play/pause controls
  • Numbered how-to instructions from exercise database
  • Pre-training notes and muscle group tags
  • Add to Workout button — updates active plan in MongoDB
  • Back navigation returns to Exercise Library or Workouts
Nutrition

Daily nutrition tracker with meal-by-meal logging. Search the USDA food database and log items to breakfast, lunch, dinner, or snacks.

  • Date navigation — view any day's intake history
  • Calorie progress bar vs. daily target
  • Meal sections: Breakfast, Lunch, Dinner, Snacks
  • USDA Food Database API for food search and kcal data
  • Add, edit, and remove food entries stored in MongoDB
Weekly Macros

A weekly overview of macro and calorie intake. Visual charts let users see patterns in their nutrition across the full week at a glance.

  • 7-day calorie summary with daily breakdown
  • Protein, carbs, and fat totals for the week
  • Progress bars vs. user-set weekly targets
  • Data aggregated from daily Nutrition logs in MongoDB
  • Highlights days where targets were hit or missed
Community

A social newsfeed where users share workouts, progress updates, and motivational posts. Modelled on a Facebook-style feed backed by MongoDB.

  • Scrollable feed of most recent community posts
  • Posts include user avatar, name, text, and timestamp
  • Filter feed by most recent or most liked
  • Coach and admin notifications on workout completions
  • Tap post to view full detail or add a reaction
Share a Post

Create and publish a post to the Community feed. Attach workout ratings, photos, or a text update. Posts are saved to MongoDB and appear instantly in the feed.

  • Text input with character limit for post body
  • 1–5 star workout rating component
  • Optional image attachment from device camera roll
  • Post saved to MongoDB with user ID and timestamp
  • Coach and friends notified via push notification
